Angular 2 CRUD application

This is an Angular 2 application that has many features you find in a lot of real world applications.

Users Posts

Features

  • Single page application (SPA) with routing.
  • CRUD operations against a backend Fake Online REST API for Testing and Prototyping - https://jsonplaceholder.typicode.com.
  • Reactive Extensions and Observables to handle asynchrony
  • Form with custom validation and dirty tracking.
  • Extracted modules for application optimization.
  • Loading content dinamically and use of master - detail views.
  • Random pictures for user comments from http://lorempixel.com
  • Filtering, pagination, dependency injection and more.

Source code: https://github.com/zaimov/angular2-CRUD
